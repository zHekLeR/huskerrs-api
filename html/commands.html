#Placeholder#'s Commands


Click each tab to see the available commands.

!stats

Returns lifetime Warzone stats for the player (not including Rebirth or Plunder).
You may also use !kd to retrieve these stats.

!weekly

Returns weekly stats for the player. You may also use !lastgames to retrieve these stats.
*If the bot hasn't been in the channel for a week yet, these may not be accurate.

!daily

Returns daily stats for the player.

!wins

Returns the number of daily wins with how many kills in each game.

!bombs

Returns the number of daily "bombs" with how many kills in each game.
A bomb is classed as a 30+ kills game but this may be subject to change on an individual player basis.

!gulag

Returns the player's daily win-loss ratio in the gulag.

!lastgame

Returns the team's performance in the previous game.

!teammates

Returns the 5 teammates that this player squads up with the most.
*If the bot hasn't been in the channel for a week yet, these may not be accurate.

!gamemodes

Returns the 5 game modes that this player queues the most.
*If the bot hasn't been in the channel for a week yet, these may not be accurate.

!matcheson

*Moderator Only*
Enables match tracking.

!matchesoff

*Moderator Only*
Disables match tracking.

!rr

Play a round of Revolver Roulette. You have 2/3 chance to survive and 1/3 chance to be timed out for 5 minutes.

!rrscore

Check your score in Revolver Roulette.

!rrscoreother [ username ]

Check someone else's score in Revolver Roulette.

!rrlb

Shows the 3 users with the most survivals in Revolver Roulette.

!rrlbdie

Shows the 3 users with the most deaths in Revolver Roulette.

!rrlbratio

Shows the 3 users with the best win / loss ratio in Revolver Roulette.

!rrlbratiolow

Shows the 3 users with the worst win / loss ratio in Revolver Roulette.

!rrtotals

Show the total survivals and deaths for all users in Revolver Roulette.

!rron

*Moderator Only*
Enable Revolver Roulette.

!rroff

*Moderator Only*
Disable Revolver Roulette.

!coin [ heads / tails ]

Guess what side the coin is on. If you lose, you will be timed out for 1 minute.

!coinscore

Check your score in Coinflip.

!coinlb

Show the user with the most wins and the user with the most losses.

!coinon

*Moderator Only*
Enables Coinflip.

!coinoff

*Moderator Only*
Disables Coinflip.

!rps [ rock / paper / scissors ]

Play Rock Paper Scissors with the bot. If you lose, you will be timed out for 1 minute.

!rpsscore

Check your score in Rock Paper Scissors.

!rpslb

Show the user with the most wins and the user with the most losses.

!rpson

*Moderator Only*
Enables Rock Paper Scissors.

!rpsoff

*Moderator Only*
Disables Rock Paper Scissors.

!bigvanish

Get randomly timed out between 100 and 1,000,000 seconds.
The timeout will automatically be removed in 3 seconds.

!bigvanishlb

Show the 3 users with the highest timeouts from Big Vanish.

!bigvanishlblow

Show the 3 users with the lowest timeouts from Big Vanish.

!bigvanishon

*Moderator Only*
Enable Big Vanish.

!bigvanishoff

*Moderator Only*
Disable Big Vanish.

!score

Calculates the current score and formats it nicely to chat.

!mc

Displays the map count to chat. This is incremented automatically as each map is added.

!setmaps [ # ]

*Moderator Only*
Sets the number of maps in the tourney.

!setplacement [ placement string ]

*Moderator Only*
Sets the placement multipliers. Example:
For these placement multipliers: 1st: 2x | 2nd-8th: 1.5x | 9th+: 1x
The format would be !setplacement 1 2 2 1.5 9 1

!addmap [ placement ] [ kills ]

*Moderator Only*
Add a map placement. The placement and kills will be saved and used later to calculate the score.
If the placement string is not correct when you add a map, just use !setplacement again. The original placement and kills and stored.

!removemap

*Moderator Only*
Remove the last added map.

!resetmaps

*Moderator Only*
Clear all of the maps.

!customson

*Moderator Only*
Enable custom tourney scores.

!customsoff

*Moderator Only*
Disable custom tourney scores.

!score

Displays the team scores in chat in the following format: 8 - 2 | Up 6
This is automatically sent to chat every 30 seconds if two vs twos are enabled and not currently paused.

!2v2on

*Moderator Only*
Enable Two vs Two scoring.

!2v2off

*Moderator Only*
Disable Two vs Two scoring.

!duel [ username ]

Challenge another user to a 50/50 duel. The loser will be timed out for 1 minute.
This has a 2 minute window for the other player to accept or deny.

!accept

Accept another player's challenge.

!coward

Reject another player's challenge.

!cancel

Cancel your challenge to another user.

!duelscore

Displays your duel wins and losses.

!duellb

Displays the 3 users with the top wins.

!duellbratio

Displays the 3 users with the best win / loss ratio.

!duelon

*Moderator Only*
Enable duels.

!dueloff

*Moderator Only*
Disable duels.
